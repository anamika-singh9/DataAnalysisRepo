# Data Analysis Project
This project demonstrates how to perform data analysis with beautiful visualisation using **Python**, **Pandas**, **NumPy**, **Matplotlib** and **Seaborn** in **Google Colab**.

## Features
- Data cleaning and preprocessing with Pandas and NumPy
- Exploratory data analysis using Seaborn and Matplotlib
- Visual insights for understanding patterns and trends
- Executed entirely in Google Colab for easy reproducibility

## Technologies Used 
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Google Colab

## How to use
1. Open the notebook
*DataAnalysis.ipynb* in Google colab.
2. Run all cells sequentially.
3. Check the visual outputs and insights generated.
